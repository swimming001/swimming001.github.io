一切从埃筛的时间复杂度开始

从朴素筛法的复杂度分析出发，逐步走入质数分布的核心规律。

1. 埃拉托斯特尼筛法 (Sieve of Eratosthenes)

找出所有不超过 NNN 的质数的经典算法：

  1. 列出从 222 到 NNN 的所有整数。
  2. 从 p=2p=2p=2 开始，划去所有 ppp 的倍数（p^2, p^2+p, \dots）。
  3. 找到下一个未被划去的数作为新的 ppp，重复直到 p>Np > \sqrt{N}p>√​N​​​。

最后未被划去的数即为质数。

2. 朴素复杂度：为什么是 O(NlogN)O(N \log N)O(NlogN) 的错觉？

初看，每个质数 ppp 需划去约 N/pN/pN/p 个数，总操作次数大约为：

∑p≤NNp=N∑p≤N1p\sum_{p \le N} \frac{N}{p} = N \sum_{p \le N} \frac{1}{p} ​p≤N​∑​​​p​​N​​=N​p≤N​∑​​​p​​1​​

如果直接用：

∑p≤N1p≈loglogN\sum_{p \le N} \frac{1}{p} \approx \log \log N ​p≤N​∑​​​p​​1​​≈loglogN

当然就得到 O(NloglogN)O(N \log \log N)O(NloglogN)。

但推导这一估计本身，就需要解析数论的支持。若没有这一结果，人们可能会误用调和级数上界：

∑n≤N1n∼logN\sum_{n \le N} \frac{1}{n} \sim \log N ​n≤N​∑​​​n​​1​​∼logN

从而错误认为复杂度为 O(NlogN)O(N \log N)O(NlogN)。

3. 精确分析：O(NloglogN)O(N \log \log N)O(NloglogN) 的出处

埃筛真正的操作次数为：

C(N) = \sum_{p \le \sqrt{N}} \left(\frac{N}{p} - p + 1\right) + \text{剩余部分}

主项为：

N∑p≤N1pN \sum_{p \le \sqrt{N}} \frac{1}{p} N​p≤√​N​​​​∑​​​p​​1​​

因此，复杂度常数直接由 质数倒数和 的渐近行为决定。

4. 质数倒数和与默滕斯第二定理

默滕斯第二定理（1874）给出了精确估计：

∑p≤x1p=loglogx+M+o(1)\sum_{p \le x} \frac{1}{p} = \log \log x + M + o(1) ​p≤x​∑​​​p​​1​​=loglogx+M+o(1)

其中 M≈0.26149M \approx 0.26149M≈0.26149 为默滕斯常数。

代入 x=Nx = \sqrt{N}x=√​N​​​，得：

∑p≤N1p=loglogN+M+o(1)=loglogN−log2+M+o(1)\sum_{p \le \sqrt{N}} \frac{1}{p} = \log \log \sqrt{N} + M + o(1) = \log \log N - \log 2 + M + o(1) ​p≤√​N​​​​∑​​​p​​1​​=loglog√​N​​​+M+o(1)=loglogN−log2+M+o(1)

因此埃筛的总操作次数为：

C(N)=N(loglogN+O(1))C(N) = N (\log \log N + O(1)) C(N)=N(loglogN+O(1))

这就是经典结论：

埃筛时间复杂度为 O(NloglogN)O(N \log \log N)O(NloglogN)。

5. 默滕斯第二定理如何得来？

默滕斯第二定理的推导正是从欧拉乘积出发：

ζ(s)=∑n=1∞1ns=∏p11−p−s,(s>1)\zeta(s) = \sum_{n=1}^{\infty} \frac{1}{n^s} = \prod_p \frac{1}{1 - p^{-s}}, \quad (s > 1) ζ(s)=​n=1​∑​∞​​​n​s​​​​1​​=​p​∏​​​1−p​−s​​​​1​​,(s>1)

取对数：

\log \zeta(s) = \sum_p \frac{1}{p^s} + \text{高阶项}

利用

ζ(s)=1s−1+γ+⋯\zeta(s) = \frac{1}{s-1} + \gamma + \cdots ζ(s)=​s−1​​1​​+γ+⋯

在 s=1s=1s=1 的极点展开，得到：

∑p1ps=−log(s−1)−C+o(1),(s→1+)\sum_p \frac{1}{p^s} = -\log(s-1) - C + o(1), \quad (s \to 1^+) ​p​∑​​​p​s​​​​1​​=−log(s−1)−C+o(1),(s→1​+​​)

再通过 Tauber 型定理（或阿贝尔求和）即可反演出：

∑p≤x1p=loglogx+M+o(1)\sum_{p \le x} \frac{1}{p} = \log \log x + M + o(1) ​p≤x​∑​​​p​​1​​=loglogx+M+o(1)

所以，

埃筛的 O(NloglogN)O(N \log \log N)O(NloglogN) 本质上等价于质数倒数和发散的速度为 loglogx\log \log xloglogx。

6. 更进一步：常数因子与默滕斯第三定理

更精细的分析表明，埃筛的实际执行次数常数与连乘积有关。

在筛法中，未被划去的数的比例近似为：

∏p≤N(1−1p)\prod_{p \le \sqrt{N}} \left(1 - \frac{1}{p}\right) ​p≤√​N​​​​∏​​(1−​p​​1​​)

默滕斯第三定理给出：

∏p≤x(1−1p)∼e−γlogx\prod_{p \le x} \left(1 - \frac{1}{p}\right) \sim \frac{e^{-\gamma}}{\log x} ​p≤x​∏​​(1−​p​​1​​)∼​logx​​e​−γ​​​​

代入 x=Nx = \sqrt{N}x=√​N​​​ 得比例约为：

2e−γlogN\frac{2 e^{-\gamma}}{\log N} ​logN​​2e​−γ​​​​

与埃筛剩余质数的计数一致。

这从另一角度印证了复杂度常数中蕴含的 e−γe^{-\gamma}e​−γ​​ 因子。

7. 从埃筛到素数定理（质数密度）

埃筛的复杂度是 π(N)\pi(N)π(N) 与 ∑1/p\sum 1/p∑1/p 共同作用的结果。

默滕斯定理虽然是中间结果，却暗示了更深的规律。

利用默滕斯第一定理：

∑p≤xlogpp=logx+O(1)\sum_{p \le x} \frac{\log p}{p} = \log x + O(1) ​p≤x​∑​​​p​​logp​​=logx+O(1)

可以推出切比雪夫估计，并最终通过复分析证明素数定理：

π(x)∼xlogx\pi(x) \sim \frac{x}{\log x} π(x)∼​logx​​x​​

也就是说，

埃筛复杂度 O(NloglogN)O(N \log \log N)O(NloglogN) 里那个双重对数，恰是质数密度为 1/logN1/\log N1/logN 的直接体现。
